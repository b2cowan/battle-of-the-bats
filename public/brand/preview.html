Two-mark system

Primary mark FL Monogram

Used wherever size permits letterforms to be legible — PWA app icons (192px+), marketing pages, app store listings, email headers, og:image. Blueprint-blue structural triangle + logic-lime accent. "HQ" secondary label (top-right).

Icon mark Chevron ">" Logic Mark

Used at constrained sizes where letterforms don't survive — browser favicon (16–64px), Android notification badge (72px), touch bar, dock icon. Pure geometry, zero font dependency. Renders identically in every context.

Size-based usage rules

Context Size Mark Rationale
Browser tab favicon 16px, 32px C — Chevron FL letterforms are illegible below ~80px
Android notification badge 72px C — Chevron Monochrome context; geometry reads, letters don't
PWA home screen icon 192px B — FL Monogram Home screen gives sufficient canvas for the lettermark
PWA splash / store listing 512px B — FL Monogram Full mark with all detail visible
Email header Any B — FL Monogram Inline image in email; brand context benefits from letterform
Marketing / og:image Any B — FL Monogram Social share previews; lettermark identifies brand
