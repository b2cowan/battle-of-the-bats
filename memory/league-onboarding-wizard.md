# League Onboarding Wizard

Date: 2026-05-16

League and Club onboarding now has a house-league-first wizard. Owners can create a draft season, add divisions, configure registration defaults, and choose whether to open the existing tournament setup wizard afterward. The normal House League create-season modal was aligned with the wizard by adding registration open/close and waiver fields.

Browser verification is still needed for the full League signup/upgrade path and optional tournament branch.
