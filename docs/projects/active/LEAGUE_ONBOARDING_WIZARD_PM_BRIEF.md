# League Onboarding Wizard PM Brief

## Proposed Functionality

League and Club customers get a guided setup wizard that helps them create their first house league season, add divisions, configure registration defaults, and decide whether they also want to create a tournament.

## Why It Matters

House league setup has more moving parts than a blank dashboard communicates. A guided first-run path helps new customers understand the right order: season first, divisions second, registration settings third, public opening later.

## Customer Impact

- Faster first successful setup for League and Club customers.
- Less risk of accidentally publishing registration before the season is ready.
- Clearer optional tournament path for organizations that run both league play and events.
- More consistent setup controls between onboarding and regular House League administration.

## Priority

High for League plan readiness. The tournament onboarding polish created a strong pattern; League onboarding should now match that quality so paid-plan users get the same level of guidance.

## Success Criteria

- A new League owner can create a draft season from onboarding without leaving the guided flow.
- Divisions and registration defaults are created from the wizard at final save.
- The dashboard shows remaining setup steps when the wizard is dismissed or partially complete.
- Choosing optional tournament setup opens the existing tournament setup wizard.
- The standard House League create-season modal includes the same core registration fields as onboarding.
