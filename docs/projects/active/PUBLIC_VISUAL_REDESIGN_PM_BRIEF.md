# PM Brief — Public Visual Redesign

## What this is
A deliberate visual elevation of the public tournament experience so it doesn't just *work* — it feels premium and memorable, like a broadcast-grade sports companion app. Mobile-first (where fans live), desktop second. No ground-up rewrite: most of the impact comes from the shared design layer, so it lands consistently everywhere.

## Why it matters
The public page is the #1 reason organizers buy FieldLogicHQ and the thing parents/fans actually touch. "It works" is table stakes; "this looks incredible and I want to show people" drives word-of-mouth, renewals, and the per-tournament install/share loop we just built.

## What changes for the user (by phase)
- **Phase A — Foundation (felt everywhere):** scores and tables stop jittering and line up like a real scoreboard; cards gain depth; bars frost over scrolling content like a native app; taps feel responsive; toggles animate. Subtle but it's the difference between "fine" and "premium."
- **Phase B — Signature mobile moments:** a "My Team" bar that follows you on game day with the live score/next-game countdown; bold broadcast scorecards with rolling score digits; standings you can read at a glance (recent-form dots, run-differential bars).
- **Phase C — Depth & identity:** a bolder hero with a countdown to first pitch, polished swipeable brackets, and team pages that feel branded to each team.
- **Phase D — Bigger bets:** one-tap shareable score/standings images (free distribution), pull-to-refresh, an offline-friendly installed app, and a desktop "control room" layout.

## Customer impact
- **Parents/fans:** an experience that feels like a real app and is genuinely delightful on game day — more time-in-app, more shares.
- **Coaches/organizers:** a public page that makes their event look professional and high-budget, strengthening the sales/renewal story.

## Priority & sequencing
Ship **Phase A first** (small effort, system-wide payoff), then the **Phase B signature moments**, then C, then D. Each phase is independently shippable and verifiable.

## Success criteria
- Side-by-side, the redesigned pages read as clearly more premium than today, on mobile and desktop, in both color modes, on branded and default orgs.
- Live score changes animate cleanly with aligned digits and no flicker.
- No regressions to theming, light mode, the mobile bottom nav, or accessibility (contrast, reduced-motion, tap targets).
- Measurable lift in time-on-page / install / share once instrumented.
