How a page says which section it opens

Each page declares its mapping in one line — the header’s help button is wired to a module + section id(s) that already exist in the guide content. No new copy is written for Phase 2; the drawer renders the same section the full guide shows.

  • Divisions → tournaments / divisions-and-pools
  • Schedule → tournaments / build-the-schedule + playoffs (two sections, shown in order)
  • Registrations → tournaments / registrations-and-teams

A single drawer lives at the layout level; every page’s “?” just tells it which section to show.
